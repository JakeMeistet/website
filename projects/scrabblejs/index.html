Year 1 Application Programming Coursework

For 50% of our Application Programming module for year 1 of my Software Engineering degree we were asked to make a game of scrabble using HTML, CSS and Javascript. For this coursework I decided to go above and beyond with my project, enabling the user to play online multiplayer using lobbys to connect to each others game. Despite the game being slightly buggy if someone leaves the lobby and some word detection doesn't always work, on the general I have managed to get a functional online multiplayer game of scrabble. I have tested this game on multiple devices, so it was a shame to see it didn't work when it came to the lecturers marking. I achieved a grade of 51/100 because the game unfortunately wouldn't play on submission, however if you do wish to see this game working you can play here.

and all the code for the project is viewable on my GitHub.
